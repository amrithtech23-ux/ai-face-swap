# 🎭 AI Powered Face Swap

A Streamlit-based web application for AI-powered face swapping in images.

## ✨ Features

- 🖼️ **Easy Upload**: Drag and drop image upload interface
- 🤖 **AI-Powered**: Leverages advanced AI models for face detection
- 🔒 **Privacy-First**: Temporary processing, no permanent storage
- 📱 **Responsive**: Works on desktop and mobile devices
- 🆓 **Free Tier**: Get started with free OpenRouter credits
